# 失恋后的那一天

![img](./assets/shilianhou.png)

学期最后一门考试在下午四点结束。朋友问我为什么没有轻松的样子，我只是笑了笑，并没有细说。 按原本的计划，这会儿我应该已经坐在去往某座城市的高铁上；可聚散由不得人，几天前，我失去了继续奔赴的理由（也就是失恋了）。辅导员还宣布封校，对正打算借酒消愁的我而言，无疑是雪上加霜。

天色暗下来后，人容易在孤独里越想越深。我开始怀疑自己以前的决定是不是太莽撞；怀疑当初的选择是否藏着冲动；也怀疑某些已经失去的东西，是不是真的走远了。 这些想法像一团湿气，把我整个人都裹进去。迷迷糊糊地瘫在床上，也许我的 2020 就要这样草草收尾了。

正胡思乱想时，手机震了一下。一位几个月前联络过的校友发来一句：“快下楼。”我带着疑惑走出北十四的大门，一个女孩迎面走来，把一杯热气腾腾的奶茶塞进我手里。

那一刻，我的心像是被什么轻轻碰了一下。

她说了几句感谢和祝福的话，我听得很安静，却有点不知该怎么回应。原来这个世界上，真的有人记得你的好，也有人在你毫无准备的时候，愿意递来一份小小的善意。 原来这么普通的我，也会被某些人惦记。

想到这里，我那点低沉竟被一杯奶茶轻易化开了。我对她说：“这大概是我跨进 2021 前，收到的最好的礼物。” 远处，几个大学生在路灯下谈天，他们的脸颊在风中冻得通红，却依然大笑着，挥手送别 2020 最后的夜晚。

人生没有排练，计划也永远赶不上变化。但生活可爱之处，也许正在于这种不确定：它让人心慌，却也会突然给你一点光。像海边的老渔夫永远不知道下一次潮水，会带来锋利的鲨鱼牙，还是藏着金币的木箱。 能做的，大概只是别太早对生活缴械。有时一杯奶茶，就足以在低谷里点亮一点火苗——不烫，却暖。

指尖的余温还在，我已不再郁闷。胸口像重新攒起一股劲儿，能和这个冬天对着干，也能安心迈进新的一年。
