# 游京

![youjing](./assets/youjing.png)

第一次来天坛，只觉空旷，天高得不像在京城。晴天碧日，松柏深沉，人们沿着长道慢慢行走，或只是坐在长椅上，把自己摊进阳光里。

入口处立着“七星石”，其中却混着一块发白的，那是清朝暗暗添上的第八块。七石本来对应北斗，后朝偏要多添一颗，仿佛不如此，就不足以昭告天下易主。

七十二连廊蜿蜒向前。尽头处，祈年殿覆着蓝色琉璃，在晴空下静静流光。我望着那叠落的屋檐，一层又一层，像一道被岁月反复描深的线。殿北有扇“古稀门”，是乾隆为省步程而开，又下诏子孙未满七十不得通行，以免失了对天的敬畏。然而，清朝皇帝大多短命，历史上真正从这门里走过的，从来只有他一人。

丹陛桥上三道长路笔直如箭，昔日的帝王从这里一步步走向高坛，向“天”汇报国运与收成。似乎只有繁复的仪式加成，他们才能说服自己：天命在我。

最令我在意的，是回音壁。掌心贴向那堵据说能让声音贴着墙远行的弧壁，儿时只在课本里见到它，不过是一张插图，此刻真的站在墙下，反倒听不见什么，只觉得天地开阔，人声渺小。

暮色已沉，我走进常去的火锅店。以前总觉得火锅必须热闹，好像只有人群围坐的时刻，才配得上那一锅翻滚的气。日子久了，一个人吃饭、远行、冒险，也成了生活寻常的模样。新来的前台小哥和我渐渐熟络，总说等老板不在要给我打折，可惜店主常坐镇门口，他只好抱歉地笑笑，转身给我悄悄多端来一碟西瓜。

锅开了，白雾裹住周身，窗外是不眠的灯火与川流的车影。我在温热的水汽里出神，遐想那些向苍天祈求秩序的君王。祭坛、长桥、殿宇——是那般宏大庄严，却也那样遥远，而店里这锅热汤、多送的水果、往来的招呼，却是真正的伸手可及。

走出店门时，夜风微凉，抬头望去，与天坛所见其实是同一片苍穹。人有时会做“受命于天”的大梦，有时却觉自己不过天地蜉蝣。可日子终究是绵延的河，不是悬在空中的庙堂。

山河千古，烟火一瞬，我的一生，就在这在宏大与细微之间往返，在这一来一回的日常里慢慢走远。
