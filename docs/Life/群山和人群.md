# 群山和人群

团建刚结束，生活又迅速被北京的节奏拉紧，几天前的松弛反而在此刻清晰起来。于是拿起笔——我怕再不写点什么，这种记忆的余温就要像水汽一样散了。

按理说，“团建”两个字本身就带着一点社交义务的味道，但同龄人相似的境遇，让几个散落各地的人，初次见面也能熟络得像一起翘课的老同学。也对，成年人的距离感多半不是来自陌生，而是心里的疲惫，若彼此都愿意喘口气，倒也能轻松靠近。

抵达大理那天是个阴雨天，浓雾把苍山吞没，洱海像张被揉皱的白纸，到处灰蒙蒙的。好在隔日天公作美，一早便云销雨霁。站在玉带云游路上俯瞰洱海，山和海都被淘洗得明彻而透亮。昨日什么都看不见，今天却豁然开朗，有些风景注定要靠一点运气。

我对那些为拍照而建的白墙打卡地实在兴致寥寥，幸好崇圣寺的千年古刹能让人真正静下来。殿堂深处的佛像金身巨大，抬头仰望时，思绪也被举高到某个更高远的地方，连烦恼都显得渺小。可惜行程紧，停久了便会被催促，成年人的旅行自由，甚至不包含步伐的速度。

晚上在古镇里玩狼人杀，这家小店的主人很年轻，见缺一人便爽快入局，他的脸上洋溢着一种自在，我羡慕那种状态。工作后少有这种情绪外放的夜晚，可以放肆争吵、辩解、拍桌大笑。折腾到半夜，一行人跑到龙龛码头，看了一个什么都看不见的日出。

那一刻，脑海里闪回两年前和朋友们爬山迎日出的夜晚——热闹总是相似的，可同行的人早就换了。

旅程将尽，我的心里却生出一份安稳。不是因为谁在等我，也不是要去哪儿报到，而是觉察到：哪怕身边的位置一个接一个地空下来，我依然能把风景看完，依然能讲出我自己的故事。

想起那夜的红眼航班，窗外，天光偶尔照亮群山，而人在城市的阴影里继续走路——光亮与自由都不常在，但偶尔一次，就足以继续往前。
