                                  什么是Markdown

                                  Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式

                                  # 一、标题

                                  在想要设置为标题的文字前面加#，#后面添加空格即可。

                                  一个#是一级标题，两个#是二级标题，以此类推，最多支持六级标题。

                                  示例：

                                  # 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

                                  # 三级标题

                                  # 四级标题

                                  # 五级标题
                                  # 六级标题

                                  # 二、字体

                                  ·斜体

                                  要倾斜的文字前后各用一个*符号包围起来

                                  ·加粗

                                  要加粗的文字前后各用两个*符号包围起来

                                  ·斜体加粗

                                  要倾斜加粗的文字前后各用三个*符号包围起来

                                  ·加删除线

                                  要加删除线的文字前后各用两个~~符号包围起来

                                  示例：

                                  *倾斜的文字*
**加粗的文字**
***倾斜加粗的文字***
~~有删除线的文字~~
                                  1
                                  2
                                  3
                                  4

                                  效果：

                                  倾斜的文字

                                  加粗的文字

                                  倾斜加粗的文字

                                  有删除线的文字

                                  # 三、引用

                                  在引用的文字前加>符号即可，在有必要的时候可以嵌套使用引用。

                                  示例：

                                  >引用的内容
>>嵌套引用的内容
>>>>>再嵌套一次(>符号越多嵌套越深)
                                  1
                                  2
                                  3

                                  效果：

                                  引用的内容

                                  嵌套引用的内容

                                  再嵌套一次(>符号越多嵌套越深)

                                  在列表内容前加上-或者+或者*符号，再加一个空格即可。

                                  示例：

                                  # 八、表格（GitHub风格）

                                  语法：

                                  |表头|表头|表头|
|--|:---:|---:|
|内容|内容|内容|
|内容|内容|内容|
                                  1
                                  2
                                  3
                                  4

                                  第二行用来分割表头和内容

                                  -符号有一个就行，多加几个也没什么影响

                                  文字默认居左

                                  右边加：符号则文字居右

                                  两边加：符号则文字居中

                                  示例：

                                  | Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |
                                  1
                                  2
                                  3
                                  4
                                  5

                                  效果：

                                  Tables Are Cool
                                  col 3 is right-aligned $1600
                                  col 2 is centered $12
                                  zebra stripes are neat $1

                                  # 十二、自定义容器

                                  标题前加上三个:符号,再加上一个空格

                                  内容结束加上三个:符号，注意，此处的:要独立成行

                                  示例

                                  ::: tip
This is a tip
:::

::: warning
This is a warning
:::

::: danger
This is a dangerous warning
:::
